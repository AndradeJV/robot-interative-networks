# Robô de interação em redes sociais


## Pacotes instalados:
  - Selenium, comando: "pip install selenium"
  

### Funcionalidades:
  - Twitter:
    - Logar
    - Realizar tweets
  
  - Facebook:
    - Logar
    - Realizar publicação textual
    - Enviar mensagem

  - LinkedIn:
    - Logar
    - Enviar mensagem, em desenvolvimento
